Comandos de html
!DOCTYPE html Indica la version del html
html Se utiliza para empezar y finalizar un programa que emplea html
head Se pone dentro de html y encima de "body" para que salga el titulo en la pestaña de arriba y conecta el html con el css
body Muestra el contenido que podrá ser visto al abrirlo en un navegador
h1,h2,... Ponen titulos y subtitulos en el body
p Ponen parrafo al texto, haciendo un salto de línea
ol Te permite crear listas ordenadas
li Te permite crear listas ordenadas dentro de "ol" y desordenadas dentro de "ul"
ul Te permite crear listas desordenadas
a href Te permite crear un hipertexto de redirige a otra parte dentro y fuera de la pagina
img src Te permite poner una imagen local o externa dentro el html
b o strong Te permite poner el texto en negrita
table te permite crear tablas
tr Te permite crear columnas
td Te permite crear filas
border Te permite poner un border a la pagina
colspan Permite juntar x filas en una misma columna
